Get the best prices for your cart in ASOS Web app, using ASOS api's

Front End - ReactJS
Back End - NodeJS

Front End:

  User enter links from asos and adding to cart in our web app
  Click Calculate
  Get his cart prices from scraped from all asos global websites with tax & available discounts calculation
  
  
Back End:

  Server get cart from front end.
  With asos API, get prices of all selected countries and return all prices + available sizes
